Buttons

The Button, an essential part of any user experience. Buttons come in four basic styles in CompUI: The button element has flat color by default, whereas .btne has a simple button, .btn success-link gives link button, .btn-icon gives Icon buttons and .floating-btn gives floating biutton.

Solid Buttons

The button component has four styles, each with its own semantic meaning. To use the button use .btn .success, .info , .warning and .danger.

Link Buttons

The link button component has four styles, each with its own semantic meaning. To use the button use .btn .success-link, .info-link , .warning-link and .danger-link .

Icon Buttons

The icon button component has different styles, each with its own semantic meaning. To use the button use .btn .btn-icon .success-icon and .danger-icon .

Floating Buttons

The button component has different styles, each with its own semantic meaning. To use the button use .floating-btn .
