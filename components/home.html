Is Comp UI For You?

You should use Comp UI if you're embarking on a smaller project or just don't feel like you need all the utility of larger frameworks. Comp UI only styles a handful of standard HTML elements and includes a grid, but that's often more than enough to get started. Love Comp UI and want to Tweet it, share it, or star it? Well, I appreciate that and I'd love to see it become a part of your project.

How To Use

Comp UI is a CSS library. This means it provides CSS classes to help you style your HTML code. To use Comp UI, you can either use the pre-compiled .css file so you can customize it to your needs.

You can also import individual components if you only need a few of them. The links to import individual components can be found in their respective pages.
