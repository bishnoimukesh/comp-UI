Image

Images in Comp UI are made responsive and round with .img-res and .img-round. This applies max-width: 100%; and height: auto; to the image so that it scales with the parent width for responsive and for round image it provides border-radius to 50%.

Responsive Image

This is a responsive image. The will will grow the image to 100% with respect to the parent container width. To use this image, add .img-res class to the image.

Round Image

This is a rounded image. The will will make the image to rounded. To use this image, add .img-round class to the image.
